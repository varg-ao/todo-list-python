# 📋 Gerenciador de Tarefas em Python

Projeto simples de linha de comando para gerenciamento de tarefas, desenvolvido em Python.

## 💡 Funcionalidades

* Adicionar tarefas
* Listar tarefas
* Remover tarefas
* Marcar tarefas como concluídas
* Salvamento automático em arquivo

## 🧠 Conceitos praticados

* Funções em Python
* Listas e dicionários
* Estruturas de repetição
* Manipulação de arquivos
* Tratamento de erros

## 💾 Como executar

```bash
python main.py
```

## 📁 Estrutura

```
main.py
tarefas.txt
README.md
```
